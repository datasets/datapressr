# Timatic database of travel requirements (visa, 

Timatic = Travel Information Manual Automatic. A list requirements required for a traveller to enter a given country. Heavily used by airlines. Maintained by IATA. https://www.timaticweb.com/

https://en.wikipedia.org/wiki/Timatic

https://github.com/tanvach/timatic-scraper has a scraper plus data from 2015.
