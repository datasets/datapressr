# Conflicts and Battle Deaths Data

Data on armed conflicts and associated deaths and injuries.

* [x] Uppsala Conflict Data Program (UCDP) dataset http://ucdp.uu.se/downloads/ - 1946-present
  * ~~1946-2008 - https://www.prio.org/Data/Armed-Conflict/Battle-Deaths/~~ - this is now  merged into Uppsala Conflict Data Program (UCDP) dataset
* [x] Correlates of War - http://www.correlatesofwar.org/ - dataset  on old datahub https://old.datahub.io/dataset/cow
* [x] Brecke dataset http://www.cgeh.nl/data#conflict
