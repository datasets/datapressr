# CLDR Common Locale Data Repository

http://cldr.unicode.org/

http://cldr.unicode.org/index/downloads

From the page CLDR includes:

1.  **[Locale-specific patterns for formatting and parsing:](http://cldr.unicode.org/cldr-features#TOC-Locale-specific-patterns-for-formatting-and-parsing:)** dates, times, timezones, numbers and currency values
2.  **[Translations of names:](http://cldr.unicode.org/cldr-features#TOC-Translations-of-names:)** languages, scripts, countries and regions, currencies, eras, months, weekdays, day periods, timezones, cities, and time units
3.  **[Language & script information:](http://cldr.unicode.org/cldr-features#TOC-Language-and-script-information:)** characters used; plural cases; gender of lists; capitalization; rules for sorting & searching; writing direction; transliteration rules; rules for spelling out numbers; rules for segmenting text into graphemes, words, and sentences
4.  **[Country information:](http://cldr.unicode.org/cldr-features#TOC-Country-information:)** language usage, currency information, calendar preference and week conventions, and telephone codes
5.  **[Other:](http://cldr.unicode.org/cldr-features#TOC-Other:)** ISO & BCP 47 code support (cross mappings, etc.), keyboard layouts
